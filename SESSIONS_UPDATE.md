# Обновление индикаторов

## Изменения

### 1. SMA индикаторы по умолчанию отключены ✅

**Файл:** `app/page.tsx`

Изменены начальные значения состояний:
```typescript
const [showSMA20, setShowSMA20] = useState(false); // было: true
const [showSMA50, setShowSMA50] = useState(false); // было: true
```

Теперь при загрузке приложения индикаторы SMA20 и SMA50 будут отключены по умолчанию. Пользователь может включить их через меню "Индикаторы".

### 2. AI знает о текущей торговой сессии ✅

**Новые файлы:**
- `lib/tradingSessions.ts` - утилита для определения торговых сессий

**Обновленные файлы:**
- `components/AIChat.tsx` - передает информацию о сессии в контексте
- `app/api/chat/route.ts` - использует информацию о сессии в анализе

#### Что добавлено:

1. **Утилита для определения торговых сессий:**
   - `getCurrentTradingSessions()` - возвращает активные сессии
   - `getPrimarySession()` - возвращает основную сессию
   - `isSessionOverlap()` - проверяет перекрытие сессий
   - `formatCurrentSessionInfo()` - форматирует информацию для отображения

2. **Торговые сессии:**
   - **Sydney** (22:00-07:00 UTC) - Низкая волатильность
   - **Tokyo** (00:00-09:00 UTC) - Средняя волатильность
   - **London** (07:00-16:00 UTC) - Высокая волатильность
   - **New York** (13:00-22:00 UTC) - Высокая волатильность

3. **AI учитывает сессию в анализе:**
   - Sydney/Tokyo - требует score >= 65, рекомендует уменьшить позицию на 50%
   - Перекрытие сессий - лучшее время для торговли, можно входить при score >= 45
   - London/New York - стандартные условия, score >= 50

4. **Контекст для AI:**
```typescript
tradingSession: {
  current: sessionInfo,
  activeSessions: currentSessions.map(s => s.displayName),
  isOverlap: currentSessions.length > 1
}
```

#### Преимущества:

- AI понимает текущую рыночную активность
- Корректирует рекомендации в зависимости от волатильности сессии
- Предупреждает о низкой волатильности в азиатскую сессию
- Рекомендует агрессивные входы при перекрытии сессий
- Учитывает особенности каждой валютной пары в зависимости от сессии

✅ Визуализация сессий убрана (не нужна на графике)
✅ Все изменения протестированы и работают без ошибок
