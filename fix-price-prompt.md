# Исправление промпта для AI

## Проблема
AI "исправляет" цену $5043 на $2504, думая что $5000+ нереально для золота.

## Решение
Добавить в начало КАЖДОГО промпта:

```
⚠️ КРИТИЧЕСКИ ВАЖНО - ЦЕНА ЗОЛОТА:
- XAU/USD = цена золота ЗА УНЦИЮ в долларах
- Текущая цена $5043 - это РЕАЛЬНАЯ цена за унцию в 2026 году
- НИКОГДА не "исправляй" цену на $2500!
- Золото может стоить $5000+ за унцию - это НОРМАЛЬНО
- Используй ТОЛЬКО указанную цену $5043 для всех расчетов
- Все уровни должны быть около $5043 (±5% = $4780-$5295)
```

## Где исправить
В файле `app/api/visual-analysis/route.ts` найти все 4 места где `systemPrompt = ` и добавить эту инструкцию в начало.

## Быстрое решение
Перезапусти dev сервер после изменений в route.ts
