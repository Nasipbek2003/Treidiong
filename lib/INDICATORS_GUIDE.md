# Руководство по техническим индикаторам

## Обзор

Система использует 10 основных технических индикаторов для анализа рынка. Некоторые индикаторы видимы в UI, другие работают в фоне и доступны только для AI анализа.

## Индикаторы

### 1. EMA (Exponential Moving Average) - Экспоненциальная скользящая средняя
**Статус:** Видимый (EMA 12/26/50)

**Назначение:** Определяет направление тренда и динамические уровни поддержки/сопротивления.

**Интерпретация:**
- Цена выше EMA → восходящий тренд
- Цена ниже EMA → нисходящий тренд
- Пересечение EMA 12 и EMA 26 → сигнал MACD

**Использование AI:** Определение краткосрочных и среднесрочных трендов, динамические уровни входа.

---

### 2. RSI (Relative Strength Index) - Индекс относительной силы
**Статус:** Видимый

**Назначение:** Показывает перекупленность и перепроданность актива.

**Интерпретация:**
- RSI < 30 → перепродан (возможна покупка)
- RSI > 70 → перекуплен (возможна продажа)
- RSI 30-70 → нейтральная зона

**Использование AI:** Определение точек разворота, дивергенции с ценой.

---

### 3. MACD (Moving Average Convergence Divergence)
**Статус:** Видимый

**Назначение:** Показывает силу тренда и возможные развороты.

**Интерпретация:**
- Гистограмма > 0 → бычий импульс
- Гистограмма < 0 → медвежий импульс
- Пересечение линий → сигнал разворота

**Использование AI:** Подтверждение силы тренда, поиск дивергенций.

---

### 4. ATR (Average True Range) - Средний истинный диапазон
**Статус:** Видимый

**Назначение:** Измеряет волатильность рынка для расчета стоп-лоссов и тейк-профитов.

**Интерпретация:**
- Высокий ATR → высокая волатильность (широкие стопы)
- Низкий ATR → низкая волатильность (узкие стопы)

**Использование AI:** Адаптивные стоп-лоссы (entry ± ATR × multiplier), оценка риска.

---

### 5. Bollinger Bands - Полосы Боллинджера
**Статус:** Видимый

**Назначение:** Показывает границы движения цены и сжатие перед импульсом.

**Интерпретация:**
- Сжатие полос → готовится импульс
- Цена у верхней границы → перекупленность
- Цена у нижней границы → перепроданность
- Пробой границы → сильный сигнал

**Использование AI:** Определение зон перекупленности/перепроданности, предсказание импульсов.

---

### 6. ADX (Average Directional Index) - Индекс направленного движения
**Статус:** Расширенный (скрыт по умолчанию)

**Назначение:** Показывает силу тренда (есть тренд или флет).

**Интерпретация:**
- ADX > 25 → сильный тренд
- ADX 20-25 → средний тренд
- ADX < 20 → флет (боковое движение)
- +DI > -DI → восходящий тренд
- +DI < -DI → нисходящий тренд

**Использование AI:** Определение торговать по тренду или во флете, фильтрация ложных сигналов.

---

### 7. Stochastic RSI - Стохастический RSI
**Статус:** Расширенный (скрыт по умолчанию)

**Назначение:** Точные входы на малых таймфреймах (скальпинг).

**Интерпретация:**
- K < 20 → перепродан (сигнал на покупку)
- K > 80 → перекуплен (сигнал на продажу)
- Пересечение K и D → подтверждение сигнала

**Использование AI:** Точные точки входа на M5-M15, подтверждение разворотов.

---

### 8. SuperTrend - Супертренд
**Статус:** Расширенный (скрыт по умолчанию)

**Назначение:** Четкие сигналы BUY/SELL по тренду.

**Интерпретация:**
- Direction = BUY → сигнал на покупку
- Direction = SELL → сигнал на продажу
- Value → уровень стоп-лосса

**Использование AI:** Простые и четкие сигналы входа, трейлинг-стоп.

---

### 9. Ichimoku Cloud - Облако Ишимоку
**Статус:** Расширенный (скрыт по умолчанию)

**Назначение:** Комплексный анализ тренда, уровней и импульса.

**Компоненты:**
- Tenkan-sen (9) - конверсионная линия
- Kijun-sen (26) - базовая линия
- Senkou Span A/B - облако (поддержка/сопротивление)
- Chikou Span - запаздывающая линия

**Интерпретация:**
- Signal = bullish → цена выше облака, Tenkan > Kijun
- Signal = bearish → цена ниже облака, Tenkan < Kijun
- Signal = neutral → цена в облаке

**Использование AI:** Определение глобального тренда, сильные уровни поддержки/сопротивления.

---

### 10. OBV (On Balance Volume) - Балансовый объем
**Статус:** Расширенный (скрыт по умолчанию)

**Назначение:** Подтверждение движения цены объемом.

**Интерпретация:**
- OBV растет → покупатели сильнее (подтверждение роста)
- OBV падает → продавцы сильнее (подтверждение падения)
- Дивергенция OBV и цены → сильный сигнал разворота

**Дивергенции:**
- Бычья: цена падает, OBV растет → скоро рост
- Медвежья: цена растет, OBV падает → скоро падение

**Использование AI:** Подтверждение силы движения, поиск скрытых дивергенций.

---

## Дополнительные индикаторы

### Volume SMA - Средний объем
**Статус:** Фоновый (для AI)

**Назначение:** Определение всплесков объема.

**Интерпретация:**
- Текущий объем > Volume SMA × 1.5 → всплеск объема (важное движение)

---

### CVD (Cumulative Volume Delta) - Кумулятивная дельта объема
**Статус:** Фоновый (для AI)

**Назначение:** Показывает баланс покупателей и продавцов.

**Интерпретация:**
- CVD > 0 → покупатели доминируют
- CVD < 0 → продавцы доминируют

---

## Как AI использует индикаторы

### Для анализа покупки (BUY):
1. **RSI < 30** или **Stochastic RSI < 20** → перепродан
2. **SuperTrend = BUY** → подтверждение тренда
3. **ADX > 25** и **+DI > -DI** → сильный восходящий тренд
4. **Ichimoku = bullish** → глобальный бычий тренд
5. **OBV дивергенция bullish** → скрытая сила покупателей
6. **MACD histogram > 0** → бычий импульс
7. **Цена у нижней Bollinger Band** → перепроданность

### Для анализа продажи (SELL):
1. **RSI > 70** или **Stochastic RSI > 80** → перекуплен
2. **SuperTrend = SELL** → подтверждение тренда
3. **ADX > 25** и **+DI < -DI** → сильный нисходящий тренд
4. **Ichimoku = bearish** → глобальный медвежий тренд
5. **OBV дивергенция bearish** → скрытая сила продавцов
6. **MACD histogram < 0** → медвежий импульс
7. **Цена у верхней Bollinger Band** → перекупленность

### Для определения стоп-лоссов:
- **ATR** → адаптивные стопы (entry ± ATR × 1.5-2.0)
- **SuperTrend value** → динамический трейлинг-стоп
- **Bollinger Bands** → стоп за границей полосы

## Отображение в UI

### Всегда видимые:
- RSI, MACD, EMA 12/26/50, SMA 20/50/200, Bollinger Bands, ATR

### Расширенные (кнопка "Показать все"):
- ADX, Stochastic RSI, SuperTrend, Ichimoku, OBV, Volume SMA, CVD

### Только для AI (не отображаются):
- Все индикаторы доступны в контексте для AI анализа

---

## Примеры использования

### Скальпинг (M5-M15):
- **Stochastic RSI** - точные входы
- **SuperTrend** - направление
- **ATR** - размер стопа

### Дневная торговля (H1-H4):
- **RSI + MACD** - сигналы входа
- **ADX** - сила тренда
- **Ichimoku** - глобальный тренд

### Свинг-трейдинг (D1):
- **EMA 50/200** - долгосрочный тренд
- **Ichimoku** - структура рынка
- **OBV** - подтверждение объемом

---

## Комбинации индикаторов

### Сильный сигнал на покупку:
```
RSI < 30 + SuperTrend = BUY + ADX > 25 + OBV дивергенция bullish
→ Очень высокая вероятность роста
```

### Сильный сигнал на продажу:
```
RSI > 70 + SuperTrend = SELL + ADX > 25 + OBV дивергенция bearish
→ Очень высокая вероятность падения
```

### Флет (не торговать):
```
ADX < 20 + Bollinger Bands сжаты + RSI 40-60
→ Ждать пробоя
```

---

## Заключение

Все индикаторы работают вместе для создания комплексной картины рынка. AI анализирует все индикаторы одновременно и выдает взвешенное решение на основе их комбинации.
